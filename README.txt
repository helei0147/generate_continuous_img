要点：
读取的mask是按照python的填充方式排序的（先行后列），而rgb中像素的排序方式是按照matlab的填充方式是先列后行的。因此，要对mask和画布进行转置后再进行填充，填充后再转置恢复为正常的方向。
get_patch.py
根据给定的材质编号、模型编号和有效patch位置获取连续光照变化的小块组合，返回的数据是[有效小块位置个数、图像高度、图像长度、光照]（四维矩阵）

在get_patch.py文件中会生成四个文件夹的信息
	1. blocks 连续光照下的小块图像
	2. info 包含小块所属模型编号和小块位置的编号（小块位置储存在其对应模型的pos_buffer文件夹中）
	3. mats 包含对应的材质编号（0~99）
	4. lights 包含对应小块的光照方向
